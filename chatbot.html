üìö Conceptos Clave

random.choice(lista)

Selecciona un elemento aleatorio de una lista. Es la funci√≥n que da variedad a las respuestas del chatbot.

Funcionamiento:
Cada vez que se llama, retorna un elemento diferente (o igual) de la lista de forma aleatoria. 
import random

saludos = ["Hola", "¬øQu√© tal?", "Buenas"]
print(random.choice(saludos))  # Imprime uno aleatorio

# Cada llamada puede dar resultado diferente:
# Primera llamada: "Hola"
# Segunda llamada: "¬øQu√© tal?"
# Tercera llamada: "Hola" (puede repetirse)

str.strip()

Elimina espacios en blanco al inicio y final de un string. Previene errores de coincidencia por espacios extras.

texto = "  hola  "
limpio = texto.strip()
print(f"'{limpio}'")  # 'hola'

# Sin strip:
if "  hola  " == "hola":  # False
    print("No coincide")

# Con strip:
if "  hola  ".strip() == "hola":  # True
    print("¬°Coincide!")

str.capitalize()

Convierte la primera letra en may√∫scula y el resto en min√∫sculas. Normaliza el nombre del usuario.

nombre = "JUAN"
normalizado = nombre.capitalize()
print(normalizado)  # "Juan"

nombre2 = "maria"
print(nombre2.capitalize())  # "Maria"

str.lower()

Convierte todo el texto a min√∫sculas. Permite comparaciones sin importar may√∫sculas/min√∫sculas.

entrada = "HOLA"
if entrada.lower() in ["hola", "hi", "hello"]:
    print("Saludo reconocido")

# Sin lower(), "HOLA" no estar√≠a en la lista
# Con lower(), "HOLA" se convierte en "hola" y coincide

Operador "in" para Listas

Verifica si un elemento existe en una lista. Retorna True si est√° presente, False si no.

saludos = ["hola", "hi", "hello"]

if "hola" in saludos:
    print("Es un saludo")  # Se ejecuta

if "adios" in saludos:
    print("Es un saludo")  # NO se ejecuta

f-strings (formatted strings)

Permite insertar variables dentro de strings de forma limpia usando f"...".
